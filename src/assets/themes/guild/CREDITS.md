# Guild theme credits

這六位角色由 [Universal LPC Spritesheet Character Generator](https://liberatedpixelcup.github.io/Universal-LPC-Spritesheet-Character-Generator/) 的多個獨立圖層組成。
各圖層可能採 OGA-BY／OGA-SA、CC-BY／CC-BY-SA、GPL 或 CC0；使用與再散布時必須依逐層條目遵守署名及相同方式分享條款。

完整且未簡化的官方產生器輸出，依角色保存在資產正本庫：

- `ui-asset-library/custom/sprites/A_sword_red/source/credits.txt`
- `ui-asset-library/custom/sprites/B_catgirl_blue/source/credits.txt`
- `ui-asset-library/custom/sprites/C_kimono_pink/source/credits.txt`
- `ui-asset-library/custom/sprites/D_paladin_blonde/source/credits.txt`
- `ui-asset-library/custom/sprites/E_sage_white/source/credits.txt`
- `ui-asset-library/custom/sprites/F_archer_orange/source/credits.txt`

總表與核對狀態：<https://github.com/alvis9315/ui-asset-library/blob/main/CREDITS.md>

`meta.json` 的 `credits_status: "complete"` 表示上述逐層署名檔已存在並核對；它不代表所有角色都有可直接重建的完整 generator hash。E 角的 hash 缺口仍明確保留。
