# 視覺公式 — workbench-nexus

> 由 9 張靈感圖抽象而成(格鬥遊戲選角畫面 ×4、像素主選單 ×2、像素 UI kit、像素頭像牆、復古選角)。
> 方法是抽公式、不抄畫面;原始參考圖檔待歸檔至 `docs/ux/references/`(使用者持有)。

## 統一風格公式

**「街機選角室」= 深太空底 × 高飽和面板 × 金黃選取 × 像素 display 字 × 網格陣列**

| 元素 | 公式 | 來源 |
|------|------|------|
| 底色 | 深太空紫黑 `#0d0a1e` + 放射漸層(星空感,不放可動背景) | 選角畫面的宇宙底 |
| 面板 | 飽和深紫卡片 `#171233` + 粗邊框 2px + 硬陰影(卡帶感) | 選角格子的金框/木框 |
| 選取態 | 街機金黃 `#ffcf3f` 邊框 + 硬陰影(hover/focus/選中一律用它) | 選角游標 |
| 標題字 | Press Start 2P(像素),只用於 display 層級 | 各圖標題字 |
| 內文字 | Noto Sans TC 無襯線(可讀性硬規則:名稱+分類恆在且可讀) | 可用性守則 |
| 版型 | 技能牆 = 選角網格 auto-fit minmax(200px,1fr);間距寬鬆不擁擠 | 選角格子牆 |
| 狀態色 | ready=綠 / running=黃 / idle=中性(Badge) | 選角 Ready 橫幅 |
| 動效 | 進場進度條、卡片 hover 微浮起;無限循環動畫禁用 | 主選單進場 |

## Keep / Avoid

- Keep:頭像+名字+分類標籤三件套、狀態一目瞭然、乾淨主選單
- Avoid:特定遊戲角色/IP(版權)、過度華麗立繪、格子過密、背景搶內容、彩虹漸層
