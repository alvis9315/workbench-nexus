# Resume Template: tw-zh-v1(台灣本土中文版)

> slug: `tw-zh-v1`。視覺層規格——內容層規則屬 resume-tailoring `resume-criteria.md` 台灣本土路線。基準:104 / Cake 簡潔版型,條列為主體、不做花式表格。

## 樣式規格(reference-doc 依此生成)

```text
版面:A4;單欄為主(基本資料區可左右兩欄簡排,但避免複雜表格)
字體:微軟正黑體或思源黑體(整份單一字體);正文 10.5–11pt;行距 1.15
姓名:16pt 粗體 + 應徵職稱一行;大頭照位(右上,佔位可留可拿——依產業慣例,
  criteria:平整正面、無風格照)
段落標題(H2):12pt 粗體 + 細分隔線;順序:
  個人簡介 / 專業技能 / 工作經驗 / 學歷 / 語言能力 / 證照 / 自傳(職缺要求才有)
工作經驗:職稱 + 公司粗體、起訖年月靠右;條列句,可比英文版稍完整
顏色:黑 + 單一深色強調(標題分隔線用),不超過兩色
頁數:1–2 頁;自傳 400–800 字(criteria 規則)
```
